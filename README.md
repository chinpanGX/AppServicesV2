# 概要

Unityで汎用的に使えるモジュール群です。

以下のパッケージから構成されています。

### UniVerse Core
サービスロケータやステートマシンの機能、MVPパターンのインターフェースを提供
※UniTaskに依存

### UniVerse ViwSystem

セーフエリアの対応するためのコンポーネント、2層のViewのレイヤー管理システムを提供
※UniVerse Coreに依存

### UniVerse Component Ex

Addressables, PlayableAPI, UGUI, の拡張機能を提供
※ UniTask, R3に依存
